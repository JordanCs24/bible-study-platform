# Bible Study Platform

A cloud-hosted Bible study web app built on AWS. Features AI-powered 
verse search, thematic exploration, and data analytics across the KJV 
and WEB translations.

## Tech Stack
- Frontend: HTML, CSS, JavaScript hosted on S3 + CloudFront
- Backend: AWS Lambda (Python) + API Gateway
- AI: AWS Bedrock
- Data: S3 data lake + AWS Athena
- CI/CD: GitHub Actions

## Phases
- Phase 1: Static frontend with verse search
- Phase 2: AI-powered verse finder and theme explorer
- Phase 3: Data analytics and visualizations
- Phase 4: User accounts, sharing, and production polish

## Data Sources
KJV Bible data sourced from [aruljohn/Bible-kjv](https://github.com/aruljohn/Bible-kjv), public domain.
